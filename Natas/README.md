# NATAS
The Natas wargame is an online game offered by the OverTheWire community. This wargame is for the ones that want to learn the basics of serverside web-security. This repository contains my writeups for the Natas wargame, a web security challenge focused on teaching basic to intermediate web exploitation techniques.

# About Natas

Natas is an online wargame that helps beginners learn:

    Web vulnerabilities
    Server misconfigurations
    Authentication bypass techniques
    Basic exploitation skills

Each level provides credentials to the next level after solving it

# Tools Used
  
    Browser DevTools
    cURL
    Burp Suite (optional)
    Python (for automation)
