# Level 0

## Goal               
Log into the server using SSH
 
## Command         
 ```
ssh bandit0@bandit.labs.overthewire.org -p 2220
```
## Explanation         
ssh(Secure shell) login is used to communicate over an unsecured network securely . Here we are connecting to a specific port number 2220
